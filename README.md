# Installation
``` 
git clone https://github.com/darklotuskdb/frida-scripts-fetcher-exploiter.git && cd frida-scripts-fetcher && chmod +x *.sh
```
```
./Install.sh
```

# Usage
```
./frida-scripts-fetcher.sh -s 'jailbreak'
```
